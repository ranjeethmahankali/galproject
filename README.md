# GeomAlgoLib
A library of computational geometry algorithms

### Dependencies

vcpkg dependencies:
* boost, boost-python, boost-program-options
* python3
* glm
* glfw3
* GLEW
* imgui
* gtest
* tinyobjloader
* efsw
* freetype
* tbb

EFSW depends on `inotify`. It can be installed as follows:
```
sudo apt-get install inotify-tools
```
